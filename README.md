# 🎬 Movie API Project (Fast & Furious Collection)

A responsive **Movie Poster Web App** built with **HTML, CSS, and JavaScript**, powered by the OMDb API. This project showcases the Fast & Furious franchise with dynamic data fetching, sorting functionality, and a clean cinematic UI.

---

## 🚀 Live Demo

👉  https://blakeurena.github.io/Movie-API-Project/

---

## 🧠 Features

* 🎬 Fetch movie data from the **OMDb API**
* 🖼️ Display movie posters dynamically
* 📅 Show movie titles, release years, and descriptions
* 🔄 Sort movies:

  * Alphabetical (A → Z)
  * Alphabetical (Z → A)
  * Newest → Oldest
  * Oldest → Newest
* 📱 Fully responsive design (mobile-friendly)
* 🎨 Clean, modern UI with hover effects
* 🧭 Smooth scrolling navigation
* 🚫 Disabled "Contact" link (UX signal)
* 🦶 Structured footer

---

## 🛠️ Tech Stack

* **HTML5** – Semantic structure
* **CSS3** – Flexbox, responsive design, animations
* **JavaScript (ES6+)** – DOM manipulation, async/await, API integration
* **OMDb API** – Movie data
